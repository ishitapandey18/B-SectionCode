HTML(Hypertext Markup Language) was created in 1990 by Sir Tim Berners-Lee.HTML is used to create web pages by defining their structure and content. The first version of HTML was written in 1993.HTML 4.O1 was the most widely version of HTML is the 2000's.It became an official standard in 1999.
HTML is used to structure web pages and tells web browsers how to display text,images and other media on a web page.
